# PDF Reader (searchable or non earchable)
## For Windows system
#### Install Tesseract OCR (tcr_install.exe exist in current dir)
#### Set up environment
```
python -m venv venv
```
#### Uncomment
```
init_tesseract()
```
#### Run python script
```
python main.py
```
---
## For Linux system
#### Install ocr
```
sh iother.sh
```
#### Install python packages
```
sh idep.sh
```
#### Run python script
```
sh run.sh
```
